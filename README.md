游民星空 wap专区 1.0 版源文件
---
目录说明：
- project/ 为项目文件
- project/模板/ 为项目后台模板文件
- source/ 为资源文件夹

默认安装[node](https://nodejs.org/dist/v8.10.0/node-v8.10.0-x64.msi),[cnpm](https://npm.taobao.org/)。  

开发：
1. cnpm install
2. gulp dev  

git：  
1. git init
2. git add .
3. git commit -m "..."
4. git remote add origin git@....git  
5. git push -u origin master
